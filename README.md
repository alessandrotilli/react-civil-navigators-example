# React CivilNavigators example

This repository shows a minimal example wrapping [CivilNavigators](https://docs.thatopen.com/Tutorials/Components/Front/CivilNavigators) from [That Open Company](https://thatopen.com/)

## Run on local machine

To run on your local machine clone the repo and install all dependencies
```
npm install
```

then start vite server with
```
npm run dev
```
